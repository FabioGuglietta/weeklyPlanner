How to use it

Weekly planner

Each time slot contains a dropdown menu. Select a project to assign it to that slot. The cell will show both the project column and the project name.

Use the × inside a planner cell to clear that slot.

Out of office

Check the box in the OUT OF OFFICE row to mark an entire day. The corresponding column becomes green.

Projects table

Click inside any project cell and type directly to add or edit a project. Press Enter to confirm or Esc to cancel the current edit.

Use the lightbulb button to highlight a project in yellow. Use the × button to delete a project.

Use the + button after the last column to add a new project column. Use the × next to a column name to remove that column.

Data storage

The app saves automatically in this browser using localStorage. Use Export JSON to create a backup and Import JSON to restore one.

Use Undo to revert the latest action.
